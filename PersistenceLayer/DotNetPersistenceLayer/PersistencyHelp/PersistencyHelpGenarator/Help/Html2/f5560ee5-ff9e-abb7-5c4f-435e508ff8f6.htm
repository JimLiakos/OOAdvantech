Parameters

storageMetadata
Type: OOAdvantech.PersistenceLayer..::.Storage
This parameter defines the metadata of new storage. Actually it is metadata from other storage that cloned in new storage.
StorageName
Type: System..::.String
Define the name of new storage.
StorageLocation
Type: System..::.String
This string used to specify the location of new storage. Maybe is a SQL server name or URL (Uniform Resource Locator). The format of string depends from the type of new storage
StorageType
Type: System..::.String
This parameter used to specify the type of new storage. Actual is the full name of Storage Provider for example MSSQLRunTime. MSSQLStorageProvider.
InProcess
Type: System..::.Boolean
Define the possibility to the objects of storage to run in process which open a session with storage. This means that if two processes try to open session with the same storage only one will gain the control. All storage types doesn't allow this possibility, some allow and others not.
