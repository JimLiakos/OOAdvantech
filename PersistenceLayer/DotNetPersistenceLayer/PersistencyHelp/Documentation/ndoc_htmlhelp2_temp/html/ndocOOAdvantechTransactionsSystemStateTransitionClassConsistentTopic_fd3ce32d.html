SystemStateTransition.Consistent Property

The Consistent property tels to the transaction system that system is in consistent state and can commit the transaction of system state transition. If you don't declare the Consistent property true and go out of the system state transition scoop then transaction system consider that system is inconsistent state and roll back to consistent state.
