ObjectCollection Methods

The methods of the ObjectCollection interface are listed below. For a complete list of ObjectCollection interface members, see the ObjectCollection Members topic.

Public Instance Methods

AddAdd an object actually you make a link between the owner of the object collection and the added object.
AddCollectionAdd objects massively.
ContainsCheck if exist the object in collection, actually if there is link between the owner of the object collection and the checked object.
GetEnumeratorReturn an enumerator to access the linked objects.
RemoveRemove an object actually you remove the link between the owner of the object collection and the removed object.
RemoveAllRemove all objects. Actually you remove all the links between the owner of the object collection and the objects that contain the ObjectCollection.
