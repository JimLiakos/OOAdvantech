StructureSet overview

Public Instance Fields

MembersDefine the Structure instance type members. The StructureSet is a collection structure instances. Structure instance has type and type has members. Therefore members define the metadata of StructureSet. The StructureSet operate as cursor you can set it at the start and you can call the MoveNext method through the end. At any time through the end you can get values of structure instance through member class.
SourceStorageSessionDefine the storage session that produce the StructureSet.
