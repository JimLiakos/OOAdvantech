StorageInstanceRef.LazyFetching Method

When a field or association of class declared as lazy fetching the persistence system does not load the value of this field or linked objects when object go to operative mode. But you can call this function to take care of loading the field value or linked object when you need that. Lazy fetching is useful in case when the data you retrieve is huge for instance images, streams.
